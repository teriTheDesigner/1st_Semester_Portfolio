Theme 02: Basic Web

02.01.01 Russian Constructivism

In the first assignment our task was to create a simple site for mobile view with html semantic markup and only a few basic tags.

"To make the text readable and legible I divided it using for example headlines, paragraphs, blockquotes or cites. "

Part of the learning was building a correct folder and file structure. I could not see the importance at the beginning, but in the future bigger projects it proved to be the key to organizing all the files.

02.02.01 History of the Micro Computer

In this project, I focused on creating a website that is easy to navigate. I used two column layout with css grid for the main content, and a header menu for local navigation between three html pages. I learned how to use the abbreviation element, ordered and unordered lists as well as local links.

At the end I validated my website using browser extension Web Developer to find my mistakes so I could correct them before the project deadline.

02.03.01 Interest Based website

To reflect on the whole theme learning, this website was much more complex. At first, I created a moodboard and a styletile to figure out how my picutres and colours fit to the chosen style. The 5 second test made on moodboard helped to get an objective opinion on what could be improved.

Sketching simple wireframes helped me to figure out the composition of the different elements and symetry. When designing the website, I kept in mind the Gestalt laws and contrast principles, as you can see in the picture.

Since the website needed to be responsive, I adjusted the grid layout and flexbox to change positions at different screen sizes using media queries in css.
