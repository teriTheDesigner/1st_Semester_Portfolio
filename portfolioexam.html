Theme 06: Portfolio Exam

Click to see

The first thing I decided was the design of the website. I had an idea of illustrating my learning progress through the Moon phases, so I added a night sky background using black and blue gradience and stars. I wanted the stars to shine so I added a simple css animation of pulsing.

Seeing my ideas in a simple XD prototype helped me to figure out the layout and structure, which you can read about in the section of Challenges.

The biggest challenge I faced was making the site responsive. I had used media queries before, but now I had problems with the overall layout using css grids.

Based on the principle of `Mobile First`, I started with the narrow view, which was quite simple. I really wanted to make a small menu on the left which would be fixed when the user is scrolling down the page. To make it work, I created an aside with navigation inside it with a sticky position.

For a better user experience, I only show the side menu on tablet view and a full screen. To make the left side completely empty for the menu, I created 2 columns in tablet view and 3 columns in the full screen view, and gave the first column the same width as the menu. After that, I positioned the content in the remaining space.

One of the things I have learned in this project was how to embed a pdf document to html.

I can imagine that there is a different and probably easier way to link all the pdf files to the website, but because I wanted to try something new I decided to create a separate html page for every assignment in pdf and embed the document there. To make it show on the whole page I defined a width of 100% and a hight in px.

As a part of linking all the pages, I have learned to use target="_blank" to open the file in a new tab.
