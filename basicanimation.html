Theme 04: Basic Animation

04.01.01 - A website with a game

In this theme, I improved my skills creating vector graphics in Adobe Illustrator, css animations and interactions in javascript.

The game's structure is build on different divs changing their visibility class based on user's click. This is done by selecting specific element by its id and using event listener to call a function which will hide/show the chosen div. The functions in javascript were also used to run animations on clicked objects to make the game more visually appealing.

Sound is an important part of any interactive game, which is why I recorded my own sounds to play when the user wins, loses, or clicks a bad or a good object. They can easily control the sound with buttons, which are connected to variables used in js functions.

04.01.02 - Paper prototype

In this assignment I created an interactive paper prototype demonstrating how my game is going to work. It was the time to make first sketches and think about the character design, its shapes, proportions and expressions in different situations.

A part of the assignment was figuring out how the objects will move on the page and what happens when user clicks on them. I came up with two different animations for a good object and for a bad one, to make them a little harder to hit.

04.01.04 - Activity diagram

In this project, I planned the user's journey on the page. at first, I needed to imagine all the different options the user has and plan every possible outcome of their decisions.

It was very important for the functionality of the game to plan what will happen on every click, which actions are happening simultaneously and how it is possible to win or lose the game.

04.02.01 - Asset list and design documentation

This project includes all the assets used in the game. Since the beginning I had a clear idea of the environment I wanted my game to be placed in, I just needed to decide on a perspective I wanted it to be shown. I knew I wanted to create a jungle setting with plants and trees in the foreground, some distant trees in the middleground and mountains in the background.

Instead of black, I decided to use dark purple for the elements in the foreground to create contrast with the light green background. All the UI elements were designed to be rounded to fit with the playful theme.
